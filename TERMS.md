# Términos de Servicio - ResiControl

**Última actualización:** Marzo 2026

## 1. Aceptación de términos

Al acceder y usar ResiControl, aceptas estar bound por estos términos de servicio. Si no estás de acuerdo, no uses el sistema.

## 2. Descripción del servicio

ResiControl es una plataforma para gestión, trazabilidad y notificación de correspondencia en conjuntos residenciales.

### Funcionalidades principales:
- Registro de paquetes recibidos
- Entrega de paquetes a residentes
- Notificaciones automáticas
- Panel de administración
- Historial de paquetes para residentes

## 3. Uso permitido

El sistema debe usarse para:
- ✅ Gestionar correspondencia del conjunto residencial
- ✅ Registrar paquetes en nombre de residentes
- ✅ Entregar paquetes a los destinatarios correspondientes
- ✅ Auditar el flujo de correspondencia

El sistema NO debe usarse para:
- ❌ Propósitos ilegales
- ❌ Acoso o vigilancia
- ❌ Recopilación de datos para terceros
- ❌ Actividades comerciales no relacionadas

## 4. Responsabilidades del usuario

### Del Administrador:
- Mantener actualizado el listado de residentes
- Configurar correctamente los usuarios porteros
- Supervisar el uso del sistema
- Atender solicitudes de eliminación de datos

### Del Portero:
- Registrar paquetes correctamente
- Verificar identidad del receptor
- Mantener credenciales seguras
- Reportar problemas técnicos

## 5. Limitación de responsabilidad

**ResiControl no se hace responsable de:**
- Paquetes perdidos o dañados física o electrónicamente
- Retrasos en la entrega
- Daños indirectos
- Problemas causados por uso indebido

## 6. Suspensión del servicio

Podemos suspender o cancelar el acceso si:
- Violas estos términos
- Usas el sistema para fines illegítimos
- Detectamos actividad sospechosa

## 7. Privacidad

El procesamiento de datos personales está sujeto a nuestra Política de Privacidad, que forma parte de estos términos.

## 8. Cambios en los términos

Podemos modificar estos términos en cualquier momento. El uso continuado del sistema implica aceptación de los nuevos términos.
